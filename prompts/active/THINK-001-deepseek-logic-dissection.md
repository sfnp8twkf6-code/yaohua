# THINK-001: DeepSeek 商业逻辑深挖

平台：DeepSeek

作用：先把选题想深，找到反常识观点、底层商业结构、争议点、生活启发。适合在写稿之前使用。

适合场景：你有一个选题，但感觉标题不够强、观点不够深、文案像资料整理。

最后更新：2026-06-27

## 可修改变量

- `【选题】`：这一条视频要讲什么。
- `【我的观点】`：你自己真正想说的话。
- `【资料】`：链接、数据、案例、新闻、截图文字。
- `【目标受众】`：例如普通上班族、小老板、自媒体人、做外贸的人。
- `【目标时长】`：建议先写 90-120 秒。

## 可复制提示词

```text
你现在不是文案助手，而是我的商业内容策划和批判型主编。

我的账号方向：
用普通人能听懂的方式，拆解商业案例、品牌出海、财经现象和商业结构。我要的是有观点、有故事、有反转、有生活启发的短视频内容，不要资料搬运，不要空泛鸡汤。

这次选题：
【选题】

我的初步观点：
【我的观点】

我已有的资料：
【资料】

目标受众：
【目标受众】

目标时长：
【目标时长】

请你先不要直接写完整稿，先帮我做深度拆解。

请按下面结构输出：

1. 表面故事
这条内容表面上在讲什么？

2. 真正值得讲的底层结构
这件事背后真正值得拆的商业逻辑是什么？

3. 观众最容易误解的地方
大多数人会把它看浅在哪里？

4. 反常识观点
帮我提炼 3 个更尖锐、更有传播力、但不虚假的观点。

5. 我的个人分析可以怎么插入
请给我 3 句带有“我为什么这么看”的表达，不要像 AI 总结。

6. 普通人的生活启发
这个案例和普通人的工作、赚钱、选择、认知有什么关系？不要鸡汤，要具体。

7. 可能的反驳
如果有人不同意，最可能怎么反驳？我该怎么回应？

8. 最有传播力的一句话
给我 5 句可以做视频大字或结尾金句的话。

9. 标题
给我 10 个标题：
- 3 个商业洞察型
- 3 个反常识型
- 2 个电影解说感
- 2 个大众好奇型
标题不能虚假夸张，不能用“震惊、炸裂、颠覆认知、太狠了”这种廉价词。

10. 文案框架
给我一版适合 90-120 秒短视频的框架：
- 开头 3 秒
- 大众误解
- 第一层反转
- 底层结构拆解
- 高潮判断
- 普通人启发
- 结尾金句

要求：
- 不要讨好我。如果我的观点不够强，请直接指出。
- 不要写成财经报告。
- 不要堆概念。
- 少用“本质上、值得注意的是、不仅是更是、换句话说、综上所述”。
- 优先给判断，其次才是表达。
```

## 使用备注

DeepSeek 适合先想深，但有时会写得偏理性、偏报告。建议先让它拆逻辑，再把结果交给 Gemini 或 Codex 改口播。
