# TITLE-001: 标题工厂

平台：Gemini / DeepSeek / ChatGPT / Claude / 豆包

作用：专门生成和筛选短视频标题。

适合场景：你对题目不满意，觉得不够吸引人、不够高级、不够有冲突。

最后更新：2026-06-27

## 可修改变量

- `【选题】`
- `【我的观点】`
- `【资料】`

## 可复制提示词

```text
你现在是我的短视频标题主编。

我的账号方向：
商业案例拆解、品牌出海、财经商业认知。标题要吸引人，但不能标题党；要有冲突，但不能虚假夸张；要让普通人想点开，又不能显得低级。

这条内容的选题：
【选题】

核心观点：
【我的观点】

文案或资料：
【资料】

请你给我生成 30 个标题，分成 6 类：

1. 反常识型：观众以为 A，但其实 B。
2. 悬念型：让观众想知道答案。
3. 商业洞察型：看起来更高级，更适合商业认知账号。
4. 电影解说型：像一个商业故事的片名。
5. 普通人启发型：让观众觉得和自己有关。
6. 克制高级型：不夸张，但有力量。

每类 5 个。

然后请你做标题筛选：

1. 选出最推荐的 5 个。
2. 分别说明它们的点击理由。
3. 标出风险：是否可能标题党、是否太专业、是否太平。
4. 最后给出你最推荐的 1 个，并说明为什么。

限制：
- 不要使用“震惊、炸裂、太狠了、颠覆认知、万万没想到”等廉价词。
- 不要制造虚假事实。
- 不要承诺文案里没有讲到的内容。
- 标题最好 12-24 个字，适合短视频封面或标题栏。
```

## 二次优化追问

```text
这些标题还是太普通。请你保留“不标题党、不虚假”的底线，但增强：
1. 冲突感
2. 画面感
3. 反常识
4. 普通人相关性

请重新给我 20 个，并把最弱的标题类型直接淘汰。
```
